NEM Twitter tip bot websocket API console
=========================================

NEM Twitter tip bot には websocket API が実装されています。
このAPIは連続的な操作を要求する為、一対一で操作を確認できるようにconsoleを作成しました。
あくまでdebug用に４時間ほどでくみ上げたプログラムなので実用しないで下さい。
